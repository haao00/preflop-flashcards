# Preflop 100bb Range Cards

iPhone向けのプリフロップ学習用フラッシュカードPWAです。

## 使い方

1. `index.html` をブラウザで開きます。
2. ハンドを見て、オープン頻度とレイズ額をスライダーで選びます。
3. `判定` で答えを確定します。
4. 正解なら約1.5秒表示してから次の問題へ進みます。
5. 不正解なら正解位置と選択位置が比較バーに表示されます。

`Deal` から、選択中ポジション内ランダム、全ポジションランダム、順番出題を切り替えられます。

ヘッダー右上の `%` からStats画面を開けます。ポジション別の正答率、ミスが多いハンド、ミスだけ練習へのショートカットを確認できます。

練習画面には成績の数字を出さず、連続正解数と獲得チップだけを表示します。正解するとチップが増え、連続正解でボーナスが伸びます。

起動時はHome画面が表示され、`学習`、`統計`、`レンジ表` を選択できます。レンジ表ではポジション別に13x13のOpen頻度を確認でき、セルや「このポジションを練習」から直接復習に移れます。

SBのみCall頻度にも対応しています。SBの練習ではCallスライダーが表示され、レンジ表では赤がOpen、緑がCall、青がFoldを表します。
4. Safariで開いた場合は、共有メニューからホーム画面に追加できます。

## デッキ形式

`Deck` パネルからJSONを編集できます。CSVインポートも可能です。

```json
[
  {
    "spot": "RFI",
    "position": "BTN",
    "hand": "AKo",
    "openFrequency": 100,
    "raiseSize": 2.5
  }
]
```

CSVの場合は次のヘッダーを使います。

```csv
spot,position,hand,openFrequency,raiseSize
RFI,BTN,AKo,100,2.5
```

同梱デッキは提供画像をもとにした `UTG/HJ/CO/BTN/SB` のRFI練習用データです。赤い面積を10%刻みのオープン頻度として扱い、UTGのA2sは40% openになっています。
